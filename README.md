# CRM Schedule Dashboard

专场排期系统原型，基于 `dabo-four-step-flow-prototype.html` 增强新增需求。

## 预览

GitHub Pages：

https://7eileen.github.io/CRM-Schedule-Dashboard/

本地也可以直接打开 `index.html` 查看。

## 本次新增

- 专场列表支持月份筛选，并保留当月已直播完专场。
- 新增“归档专场”板块，表达每月 5 号归档上月专场的方案。
- 达人专场录入支持飞书文档链接说明、标准产品选项、状态、类型、负责商务、排期时间等字段。
- 按目标销售额自动计算达人等级和拍剪条数建议。
- 新增内容需求单字段、剪辑支持字段、投放预估销售额字段。
- 我的任务支持按达人等级和交付时间做紧急程度排序，并增加组长/管理视角。
- 数据中心增加成员承接、专场达标率、投放预估达标率三块看板，并标注商务可见权限。
