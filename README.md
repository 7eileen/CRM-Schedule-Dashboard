# CRM Schedule Dashboard

CRM 排期管理工作台是一个零依赖单页原型，复刻了四步流程型运营工作台的视觉结构，并替换为 CRM 客户排期、派工执行、任务看板和复盘归档场景。

## 本地预览

直接打开 `index.html`，或在仓库目录启动任意静态服务：

```bash
python -m http.server 4173
```

然后访问 `http://localhost:4173`。

## 页面模块

- 排期总览：客户排期卡片、统计卡、风险提醒、节点动态
- 需求录入：客户 brief 识别、排期字段确认、资源需求开关
- 智能派工：模块负载、待派工需求、负责人建议
- 执行看板：按待处理、进行中、延期风险、已完成分栏
- 复盘归档：成交、线索、成本和 CRM 回写模板
