# CRM Schedule Dashboard

该仓库用于存放 `dabo-four-step-flow-prototype.html` 的 1:1 复刻版本。

## 预览

直接打开 `index.html` 即可查看页面。

## 说明

- 页面内容、功能模块、数据结构和交互逻辑保持源 HTML 一致。
- 当前提交不再改写原型内的业务内容。
- 仓库名称沿用 `CRM-Schedule-Dashboard`，但页面本身按用户提供的源文件复刻。

## 当前增量

- 专场列表增加排期月份筛选。
- 当月已直播完的专场仍保留在当月列表中展示。
- 专场列表增加月度日历视图，用于观察当月排期节奏。
- 达人专场录入的“专场信息”调整为指定填写项，并将主推产品改为标准下拉及可收起商品展示栏。
- “背景、紧急程度、次推产品、次推机制”不再作为达人专场录入和项目详情里的系统展示字段。
- 新建专场卡片等级改为按预估销售额自动计算，商务端不需要填写紧急程度。
- 数据中心增加“成员承接与达标率”“不同专场达标率”“投放预估达标率”三个数据板块，并通过查看身份区分商务可见范围。
